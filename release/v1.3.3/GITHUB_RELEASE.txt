FPVGate v1.3.3 - Modern Configuration UI and WebSocket Stability

Release Date: December 10, 2024

WHAT'S NEW

Modern Configuration Interface
- Complete redesign with full-screen overlay modal
- Sidebar navigation with 6 organized sections:
  * Lap & Announcer Settings (merged race setup + TTS)
  * Pilot Info (band, channel, name, callsign, color)
  * LED Setup (presets, colors, speed, brightness)
  * WiFi & Connection (mode, COM port, credentials)
  * System Settings (battery, theme, OBS, firmware)
  * Diagnostics (self-test)
- Mobile responsive with horizontal tabs on small screens
- Click-outside or ESC to close
- Inspired by Mainsail Interface settings

WebSocket Stability
- SSE keepalive ping every 15 seconds
- Fixes RSSI disconnection requiring page refresh
- Reliable long-running sessions

Bug Fixes
- Fixed duplicate element IDs causing JavaScript issues
- Fixed battery monitoring toggle
- Fixed LED animation speed visibility
- Fixed WiFi configuration Apply/Reset buttons
- Removed stray backtick-n characters

INSTALLATION

For ESP32-S3 (8MB Flash):

Required files:
- FPVGate_v1.3.3_ESP32S3_bootloader.bin (at 0x0000)
- FPVGate_v1.3.3_ESP32S3_partitions.bin (at 0x8000)
- FPVGate_v1.3.3_ESP32S3_firmware.bin (at 0x10000)
- FPVGate_v1.3.3_ESP32S3_littlefs.bin (at 0x410000)

Using esptool:
esptool.py --chip esp32s3 --port COM# write_flash 0x0000 FPVGate_v1.3.3_ESP32S3_bootloader.bin 0x8000 FPVGate_v1.3.3_ESP32S3_partitions.bin 0x10000 FPVGate_v1.3.3_ESP32S3_firmware.bin 0x410000 FPVGate_v1.3.3_ESP32S3_littlefs.bin

Using PlatformIO:
pio run -e ESP32S3 --target upload --upload-port COM#
pio run -e ESP32S3 --target uploadfs --upload-port COM#

Upgrading from v1.3.x:
- Upload just filesystem via OTA at /update
- Or use: pio run -e ESP32S3 --target uploadfs

COMPATIBILITY
